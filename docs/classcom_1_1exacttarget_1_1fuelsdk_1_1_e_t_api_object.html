Detailed Description

An ETApiObject represents an object available via the Salesforce Marketing Cloud API. All API objects are guaranteed to have the following properties, which is enforced via abstract getter and setter methods:

  • id

Typically, a specific API object will extend either ETRestObject or ETSoapObject (depending on whether it is a REST or SOAP API, respectively), adding object-specific property getters and setters. In a few cases (e.g., ETDataExtension), API objects are hybrids of both REST and SOAP.
