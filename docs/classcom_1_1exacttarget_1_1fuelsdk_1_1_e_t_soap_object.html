ETSoapObject has additional retrieve methods that take the SOAP object name and request ID for continuing requests. In most cases, the SOAP object name is the same as the internal class name, but in a few cases, it is not (e.g., DataExtensionObjects require the name of the data extension in brackets: DataExtensionObject[foo]).
